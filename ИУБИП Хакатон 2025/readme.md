# Хакатон 72 часа. Первое место на **Middle** кейсе
### Краткое описание
СЕРВИС ДЛЯ РАССЫЛКИ И ПОДПИСАНИЯ ДОКУМЕНТОВ
50к рублей
Сервис для рассылки документов сотрудникам и подписания этих документов внутри
компании.  Можно создавать предприятие с отделами и добавлять в него зарегистрированных сотрудников (в предприятии много сотрудников,
у сотрудника может быть несколько предприятий). У каждого сотрудника есть возможность отправлять другим сотрудникам документ на подписание и подписывать входящие документы.

Подпись документа должна генерируется основываясь на номере телефона сотрудника.
Подтверждение подписания через почту
